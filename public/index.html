녹화·음성을 검색 가능한 글로 바꿔요

회의 녹화, 업무 영상, 음성 메모를 올리면 카카오 GPU가 음성을 받아쓰고 시간대별 검색 결과를 만듭니다. GPU 서버만 따로 만드는 기능은 제공하지 않습니다.

  1. 1
    파일 올리기Work Memory에서 MP3·WAV·MP4 등을 선택해요.
  2. 2
    분석 작업 만들기언어를 선택하면 실행 대기 작업이 생겨요.
  3. 3
    비용 확인 후 실행예상 최대비용을 보고 동의한 경우에만 GPU를 만들어요.
  4. 4
    결과 검색완료되면 문장을 검색하고 해당 영상 시점으로 이동해요.
