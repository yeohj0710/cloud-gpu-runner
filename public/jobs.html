GPU 실행 설정

분석을 시작할까요?

GPU가 생성되는 순간 카카오클라우드 비용이 발생합니다. 작업 완료·실패 시 인스턴스 삭제를 요청합니다.

사용 방법

  1. 녹화·음성 파일을 직접 올리거나 목록에서 선택합니다.
  2. 주 언어를 고르고 분석 작업을 만듭니다.
  3. 작업 카드에서 GPU 분석 시작을 누릅니다.
  4. 완료되면 결과 보기에서 텍스트를 검색합니다.

GPU 네트워크가 준비 중이면 작업을 먼저 등록해두고 나중에 시작할 수 있습니다.
