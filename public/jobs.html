1. 파일 준비

학습 코드와 데이터를 올려주세요

코드는 ZIP 또는 tar.gz로 묶어주세요. 데이터는 CSV, JSONL, ZIP 등 형식 제한이 없습니다.

파일은 네이버 Object Storage에 저장되고, 카카오 GPU가 실행할 때만 내려받습니다.

2. 실행 설정

평소 터미널에서 쓰는 명령을 넣으세요

TIPS 연구에는 이렇게 쓰는 게 맞아요

현재 `npm run rnd:train:all`은 CPU용 TypeScript 학습이라 GPU 이득이 없습니다. 실제 사용자 데이터로 PyTorch·XGBoost GPU 학습 코드를 만들었을 때 이 작업실에 올리면 됩니다.

3. 실행 전 비용

최대로 나갈 금액을 먼저 확인해요

최대 예상 비용 · VAT 별도계산 중
언제 과금되나요?

GPU 생성부터 삭제까지의 시간 + 디스크 + 작은 저장소 요청 비용입니다. 작업이 일찍 끝나면 실제 비용도 줄어듭니다.
